# Sales Forecasting & Business Performance Analytics

## Overview
This project focuses on analyzing business sales data and forecasting future revenue trends using Python and SQL.  
The objective is to support data-driven decision-making by identifying key performance patterns across regions and products and predicting future sales.

---

## Tech Stack
- **Python**: Pandas, NumPy, Matplotlib, Statsmodels  
- **SQL**: SQLite  
- **Forecasting Model**: ARIMA  
- **Dashboard Output**: CSV files (BI / Excel / Power BI ready)

---

## Project Workflow
1. Generated a realistic synthetic sales dataset (daily data).
2. Stored and queried data using SQL for business insights.
3. Performed exploratory data analysis (EDA) in Python.
4. Built a time-series forecasting model to predict future sales.
5. Exported clean, dashboard-ready datasets.

---

## Key Analysis Performed
- Monthly revenue trend analysis  
- Region-wise sales performance  
- Product-wise revenue contribution  
- Time-series sales forecasting (next 6 months)
